About PDF Merge Tool

This web application allows you to easily merge multiple PDF files into a single document. Built with Django and PyPDF2, it provides a simple drag-and-drop interface for combining your PDFs.

Features

  • Drag and drop multiple PDF files
  • Click to browse and select files
  • Real-time file validation
  • Automatic download of merged PDF
  • File size display
  • Remove files before merging

How to Use

  1. Drop PDF files onto the upload area or click to browse
  2. Review the selected files in the list
  3. Remove any unwanted files using the Ã— button
  4. Click "Merge PDFs" when you have at least 2 files
  5. The merged PDF will automatically download
